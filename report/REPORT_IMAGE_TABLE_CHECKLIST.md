## Word Formatting Tips

1. Use Heading 1 for chapter titles and Heading 2 for sub-sections.
2. Use automatic table of contents in Word.
3. Use page break before each chapter.
4. Add figure captions under every inserted image.
5. Update page numbers after final alignment.
