# Customer Segmentation Platform - Final Year Project Report Pack

## How To Use This Pack in Word

1. Open each file in order.
2. Copy content file-by-file into a single Word document.
3. Insert page breaks at each major chapter heading.
4. Replace image placeholders with your actual screenshots/diagrams.
5. Update page numbers after final formatting.
